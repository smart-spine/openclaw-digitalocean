# OpenClaw DigitalOcean Template

Use the quickstart:

- `docs/digitalocean-git-nodocker-quickstart.md`

This template provides:

- no-Docker deployment on DigitalOcean
- supports separate user-owned public deploy repo (`REPO_URL`) with auto-sync
- one-command install: `./do.sh install`
- one-command update: `./do.sh update --to <TAG>`
- rollback/status/logs/tunnel/destroy commands
